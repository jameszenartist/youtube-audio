# Youtube Audio CLI Tool

Command line tool written in Node.js to download Youtube audio files.


More projects like these can be found at https://syntaxsamurai.com

## Usage

```
npm install

npm link

# w/ linux:

sudo npm link
```

## Commands

```
# Help & Commands
youtube-audio -h

# Version
youtube-audio -V

# API Key Commands
youtube-audio

**MORE COMING SOON**


```
